📲 Instalar la app

Tu navegador no mostró el instalador automático. Instálala manualmente así:

  • Chrome (Android/Samsung): toca el menú ⋮ (arriba a la derecha) → “Instalar app” o “Agregar a pantalla principal”.
  • Samsung Internet: menú ☰ → “Agregar página a” → “Pantalla de inicio”.
  • iPad / Safari: botón Compartir ⬆️ → “Agregar a inicio”.

Aparecerá el ícono 🍽️ y abrirá en pantalla completa.
